Homework 2

Find the IP of the (hidden) virtual machine.

You will need to configure the machine with your data, on port 80: name, surname and group. This will generate the flags.

This is a necessary step before actual doing the exercises.

All exercises except number 1 and bonus can be found in /execises/$NUMBER.

The archive uploaded should contain the following files:

config - contains on each line your data: name, surname, group (as configured)

flag_X - where x is 1,2,3,4,5,6,7,bonus

README - where you explain what you did to solve it

Any scripts or source code of binaries used to get the flags

Exercises:

1. Get shell on the device :). Look for the correct page in a webserver and break your way in. The hints provided should help you.

2. A custom encryption mechanism has been used to hide a secret. Break the encryption and find the flag !

3. Binary check_flag can be used to verify if a flag is correct. How much time would it take to brute-force a 32 character flag ? Can you do it faster than blind brute force?

5. If you can control the flow of the binary ex5, you will be able to read the flag !

6. The vulnerable program ex6 contains a call to a method that can read the flag. Figure out how to call this method and get the flag.

7. Figure out how to exploit the vulnerable program ex7 and gain shell as user flag7. Do this and you'll be able to read the flag
