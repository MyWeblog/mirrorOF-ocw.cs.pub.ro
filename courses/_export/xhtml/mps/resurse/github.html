Git și GitHub

Dezvoltarea proiectelor de MPS se va realiza pe GitHub. Fiecare echipă își va crea un repository GitHub și resursele asociate: wiki, issue tracker. Codul scris va fi privat fiecărei echipă. Aici fiecare persoană își va dezvolta modulul prin consultare cu ceilalți membri ai echipei.
