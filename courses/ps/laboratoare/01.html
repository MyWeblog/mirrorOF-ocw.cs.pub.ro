Laboratorul 01.

Lab 1 signal processing - introduction to MATLAB.

In this lab we'll get used to MATLAB (in case you forgot…) and make some very basic signal processing. Most of the signal processing techniques can be performed as operations on vectors and matrices, so MATLAB is a very useful tool to perform these operations.

Please open the tutorial available here, which we shall use as a reference.

Following chapter 1 (Quick Start), let's do the following exercises.

1. Get used to the environment: folder path, workspace, history, script edit, console.

  • type a = 2
  • type b = 3
  • type a+b
  • type c=a+b
  • type c=a+b; %(notice the semicolon ';'). Do you see any difference between the commands above ?
  • type a*b; %how can you find the answer ? (hint: variable 'ans')

2. Vectors

Signals can be represented as sequences of values, which in MATLAB is usually done through vectors and matrices. Let's start with vectors.

To generate a vector of some given values, simply type something like the following:

  • vec1 = [1, 2, 3]; % Note the brackets '[' and ']' needed to enclose values
  • Create a vector of 7 ones (with function 'ones') and then multiply it with some constant. Show the result.
  • Create a vector of 5 random values using the function rand. Type 'help rand' to find how to use it.

3. Matrices

The matrices are the two-dimensional extension of vectors, and you should already be well familiar with these from the other courses.

  • Create a matrix of 5×4 of random numbers (using the same function rand as before). Then create a matrix of ones of the same size. Now add the two matrices and show the result.

4. Transposes

The transpose of a matrix can be obtained by using the operator `'`.

  • Transpose the matrix you created above.
  • Create a column vector of ones, using the function 'ones' with a number of elements greater than one on the first dimension.

Print this vector.

  • Now transpose this vector and print it again. What is the difference in the output ?

When you deal with vectors (adding, multiplying, etc.) you will need to take care to use the proper form (either column or row vector).

5. Other ways to create arrays and matrices

  • To create a continuous sequence (say from 1 to 5), you can use the colon operator (`:'), as in v = 1:3 or v = [1:3].

Create a sequence of 10 elements, starting from 5.

  • You can make the increment different than 1, and even a non-integer value, using something like v =1:0.5:5. Try it and see the result.
  • You can concatenate vectors and matrices within brackets [], e.g. a = [1, 2, 3; 4, 5, 6] or m = [a; b] where a and b are

row vectors of the same length. Create three row vectors of the same length and concatenate as above to create a matrix.

  • Find which function you can use to determine the length and size of vectors and matrices, respectively.

6. Saving and loading variables

Check the functionality of the functions 'save' and 'load' (using 'help load', 'help save').

  • Save the variables you have created into a file.
  • Clear all your variables using 'clear';
  • Load again your variables from the file above.
  • Generate a random matrix of size 10×10. Save only this matrix into a file, but none of the other variables.
  • Clear all the variables.
  • Load only the matrix.

7. Strings and display

You can easily create strings using something like s = 'Hello'.

  • Create a variable holding the string 'Signal processing'.
  • Use the disp function to print that string
  • Now try to use the disp function to print a string that has 'Lab of ' concatenated with the previous string.
  • Now use the function fprintf or sprintf to print a string followed by some digit of your choice.

8. Functions

As in most languages, a nice property is the ability to call other functions.

  • Generate a random vector using rand
  • Find the maximum (use the function max)
  • Find the minimum

9. Plots

Plots are one of the coolest thing in MATLAB, as they allow you to visualize your data.

  • Use the function 'sin' to generate a sinewave of frequency 1 Hz over 1 second (so you should get one full period). For this you should first generate the input over [0 … 1] in very small steps (e.g. 0.01) and then call the function sin(2*pi*f*t) over this sequence (where f is the frequency, t is the input sequence and pi is the number 3.1415…) to obtain the sinewave as a sequence.
  • Check the 'plot' function using either 'help plot', or even better 'doc plot'
  • Plot the sinewave you made earlier using the plot function
  • Change the line color of the sinewave
  • Change the line type of the sinewave
  • Create another sinewave with a different frequency (e.g. 2 Hz)
  • Plot both sinewaves on the same plot (for this you should pass the sequences as a matrix containing the first sinewave on the first row and the second as the second row)
  • Try adding x-axis and y-axis labels as well as a title to your figures
  • Add a legend to clarify which signal is which

10. Some basic signal processing It's best to make your MATLAB programs in scripts, so that you can reuse them later. In this manner you can also create functions of your own. For now, let's just create some scripts.

  • Create a new script and name it 'basic_dsp.m', in which you will type the commands for the following operations.

After you have some commands in your script you can execute it to see its output, or you can execute only part of it (either by selecting some lines, commenting others – with % –, or using sections – via %% –). All of the things below should go into your script.

  • Generate two sinewaves as above with different frequencies, but spanning the same interval (e.g. 1 Hz and 2 Hz over 1 second)
  • Generate a signal having the sum of the two sinewaves
  • Plot all three signals and check the differences. Hint: use 'figure' in front of each plot command to generate three different plots or plot them all together in the same plot (or in two plots, one for the initial sinewave, one for the resulting sinewave).

You should get something like this: Un exemplu de cod este acesta:

basic_dsp.m
close all;
f1 = 1;
f2 = 2;
x = 0:0.01:1;
s1 = sin(2*pi*f1*x);
s2 = sin(2*pi*f2*x);
h1 = figure;
plot(x, [s1; s2]);
xlabel('Time (s)');
ylabel('Amplitude');
title('Two sinewaves');
legend('1 Hz', '2 Hz');
h2 = figure;
c = s1 + s2;
plot(x, c);
xlabel('Time (s)');
ylabel('Amplitude');
title('Sum of the two sinewaves');
print(h1, '-dpng', 'sine1.png');
print(h2, '-dpng', 'sine2.png');

11. A small application

In the file data.mat you are given three images, represented as three-dimensional matrices (one matrix per color - R, G, B). The matrices represent one image (IR) that has been distorted by adding some noise (matrices R1 and R2), via the formula: IR = I*0.3 + R1*0.3 + R2*0.3

Your task is to reconstruct the original image matrix and display it. For this you should:

  • load the matrices
  • display the three given images (matrices) to visualize them, using the method 'image', e.g. 'image(R1)'.
  • Subtract somehow the noise matrices
  • Display the image after subtracting the noise

Note that an image matrix only contains values from 0 to 255. Un exemplu de cod este acesta: 

II = IR - R1*0.3 - R2*0.3;
image(II)
